# 工业网络规划经验分享框架

一个面向工业网络规划场景的经验表达、复用与迭代框架。

该项目用于将工业网络规划中的资料采集、判断依据、设计推演、结果落地与经验迭代组织为统一的结构化流程，便于项目复盘、经验沉淀与后续复用。

## 本地运行

```bash
npm install
npm run dev
```

默认访问地址：

```bash
http://localhost:5173/
```

## 生产构建

```bash
npm run build
```

构建产物位于：

```bash
dist/
```

## GitHub Pages 发布

本项目已配置 GitHub Pages 自动发布。

当代码推送到 `main` 分支后，GitHub Actions 会自动执行构建并发布页面。

预计访问地址：

```bash
https://openmaxw.github.io/Industrial-Network-Planning-Framework/
```
