# 交付检查清单

## 1. 平台骨架
- 已具备首页、左侧固定骨架菜单、顶部经验加载与导入导出入口。
- 左侧骨架固定为：资料采集、判断依据、设计推演、结果落地、经验迭代。
- 首页支持按标准理论、应用场景、系统对象、技术方向组织经验入口。

## 2. 标杆经验体
- `ISA95 + IEC62443` 已作为标杆经验体具备完整五层结构。
- 标杆经验体已覆盖资料采集、规则、推演、输出、案例、内部复盘与经验迭代。

## 3. 结果落地
- 已具备面向客户的输出页。
- 已具备面向内部的规划复盘与案例提炼页。
- 输出页支持摘要、记录汇总、文本总结三类结构。

## 4. 经验元数据
- 首页已展示经验说明、适用对象、标签、状态。
- 经验目录已支持经验标签、适用对象、状态等最小元数据。

## 5. 经验扩展能力
- 当前已存在 5 套可加载经验：
  - ISA95
  - IEC62443
  - ISA95 + IEC62443
  - 储能项目
  - MES
- 系统支持外部经验导入、当前经验导出、模板下载。

## 6. 经验迭代能力
- 已增加独立顶级菜单：经验迭代。
- 已具备：特殊判断、经验候选、迭代总结。
- 设计推演页已开始显式支持推演回流字段。
- 迭代总结页已可自动汇总关键推演页中的回流标记。

## 7. 导入校验
- 已校验 `meta`、`navigation`、`pages`、`fields`、`outputs`、`cases` 基本结构。
- 对缺失字段、结构错误、数组/对象类型错误已有提示。

## 8. 构建验证
- 已执行 `npm run build`。
- 当前代码可完成生产构建。

## 9. 当前结论
当前系统已具备：
- 稳定的平台骨架
- 可复用的经验配置机制
- 多套内置经验体
- 外部经验导入导出能力
- 面向交付与沉淀的结果页
- 面向经验回流的经验迭代层

可判断为：**已达到当前目标下的可交付版本。**
