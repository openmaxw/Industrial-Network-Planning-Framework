# 工业网络规划知识表达框架

## 项目定位
本项目不再只是一个“经验驱动框架”，而是进一步演进为一个面向工业网络规划领域的**经验沉淀与协作平台**。

它的目标是：
- 让不同经验配置可以被结构化表达
- 让不同 FAE 的经验可以被沉淀、复用、比较和讨论
- 让标准理论、应用场景、系统对象、技术方向等不同角度的方法论都能在同一平台中组织与协作

## 核心原则
- 平台固定，经验配置可变。
- 首页属于平台层，经验页面属于内容层。
- 菜单、页面、字段、依赖关系、案例和输出映射由经验配置驱动。
- 平台支持经验配置导入、导出、比较和持续演进。
- 左侧导航用于当前经验内部导航，顶部角度入口用于平台级分类浏览。

## 当前已落地的架构分层
- `src/app/`：平台壳层，负责整体布局、平台入口与模块装配。
- `src/components/`：页面组件，当前包含首页、记录页、输出页等。
- `src/methodologies/`：经验配置库，当前已拆出多套经验配置。
- `src/engine/`：经验配置解析与校验逻辑。
- `src/store/`：运行时状态管理。
- `src/themes/`：平台样式。
- `docs/`：平台与经验文档。

## 当前已实现的能力
1. 平台首页
2. 平台默认介绍区
3. 顶部四个角度入口
4. 首页按角度切换内容区块
5. 内置经验选择
6. 外部经验配置 JSON 导入
7. 当前经验配置 JSON 导出
8. 当前运行数据 JSON 导出
9. 经验驱动的左侧菜单
10. 统一记录页模板
11. 输出页模板
12. 经验配置基础校验

## 当前平台视角
当前首页已按以下四个角度组织经验入口：
- 标准理论角度
- 应用场景角度
- 系统对象角度
- 技术方向角度

这意味着平台已经开始承载“不同视角的经验组织方式”，而不只是单一业务系统菜单。

## 当前经验配置状态
当前已存在多套最小经验配置入口：
- `ISA95`
- `IEC62443`
- `ISA95 + IEC62443`

其中综合经验体已开始承载旧系统页面迁移样板。
